# Task 001 - Project Bootstrap

## Objective

Initialize the AI Vocal Coach web application using modern frontend technologies.

This task should only create the project foundation.

No pitch detection or AI features should be implemented in this task.

---

- React
- TypeScript

The project must include:

- ESLint
- Proper folder structure
- Component architecture
- Responsive layout foundation
